Тип Префикс Описание
Компонент (Block) b- Формируют основу приложения и содержат всю косметику для отдельных блоков
Модуль макета (Layout module) l- У этих модулей нет никакой косметики, используются только для позиционирования блоков и для построения макета приложения.
Помощники (Helpers) h- Вспомогательные классы
Скрипты (JavaScript) js- Они указывают на то, что поведение JavaScript привязано к блоку. Стили с ними не ассоциируются; используются только для облегчения манипуляций со скриптами. Не имеют стилей
Тестирование (quality-assurance) qa- Используются для проведения тетов.Не имеют стилей
