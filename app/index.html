Элементы форм

По умолчанию размер элемента зависят от контекста в котором используется, для получения требуемой ширины ее нужно задать обертке.

  <div class="b-form-section">
    <label class="b-form-section__title" for="input-01">Input`s title</label>
    <div class="b-form-section__elem-wrapp">
      <input type="text" placeholder="Some placeholder" id="input-01">
    </div>
  </div>

Любой элемент(ы) формы должен(ы) быть заключен в тег с классом .b-form-section. Отображение дочерних элементов будет зависить от его модификации.

This field is required 
  <!-- элемент обязателен для заполнения -->
  <div class="b-form-section b-form-section--required">
    <label class="b-form-section__title"  for="input-02">Input`s title</label>
    <div class="b-form-section__elem-wrapp">
      <input type="text" placeholder="Some placeholder" id="input-02">
    </div>
  </div>

  <!-- успешная валидация -->
  <div class="b-form-section b-form-section--success">
    <label class="b-form-section__title" for="input-03">Input`s title</label>
    <div class="b-form-section__elem-wrapp">
      <input type="text" placeholder="Some placeholder" id="input-03">
    </div>
  </div>

  <!-- ошибка при валидации поля -->
  <div class="b-form-section b-form-section--error">
    <label class="b-form-section__title" for="input-04">Input`s title</label>
    <div class="b-form-section__elem-wrapp">
      <input type="text" placeholder="Some placeholder" id="input-04">
    </div>
    <div class="b-form-section__error-descr">
        This field is required
    </div>
  </div>
